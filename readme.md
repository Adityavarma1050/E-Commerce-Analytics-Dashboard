# 🛒 E-Commerce Analytics Dashboard (Power BI)

## 📌 Overview

This project is an **interactive E-Commerce Analytics Dashboard** built using **Microsoft Power BI**. It provides insights into sales, profit, orders, discounts, and customer behavior across different states and categories.

The dashboard helps businesses make **data-driven decisions** by visualizing key performance metrics in an intuitive format.

---

## 📊 Key Features

* 📍 **State-wise Sales & Profit Analysis**
* 📈 **Daily Sales Trend Visualization**
* 🧾 **Order Status Distribution (Delivered, Returned, Cancelled)**
* 🛍️ **Category-wise Sales Breakdown (Electronics, Clothing, Grocery)**
* 💰 **KPIs (Key Performance Indicators):**

  * Total Sales
  * Total Orders
  * Total Profit
  * Average Discount

---



## 🧠 Insights You Can Derive

* Identify **top-performing states** (e.g., Telangana, Maharashtra)
* Analyze **sales trends over time**
* Understand **customer order behavior**
* Compare **category performance**
* Track **profit vs sales relationship**

## ⚙️ Tools & Technologies Used

* **Power BI Desktop**

## 🔧 Steps to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. If needed, reconnect the dataset (`ecommerce_data.csv`)
4. Explore the dashboard interactively

---

## 🟡 Data Preparation Steps

* Cleaned dataset in Power BI:

  * Set correct data types (Date, Number)
  * Removed null/empty values
* Created calculated measures using DAX
* Built relationships (if applicable)
